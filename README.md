# review_ratings_api

The Review & Ratings API is a micro-service for an ecommerce application. The service has been scaled to handle a minimum of 250 requests per second with a response time of 14ms.

### Technologies Used

* [Express](http://expressjs.com/)
* [PostgreSQL](https://www.postgresql.org/)
* [Redis](https://redis.io/)
* [Nginx](https://www.nginx.com/)
* [Artillery](https://artillery.io/)
* [Loader.io](https://loader.io/)
